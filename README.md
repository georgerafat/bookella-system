# نظام Bookella لإدارة المكتبة والمخزون 📚

هذا المشروع هو نظام متكامل (ERP) مصمم خصيصاً لمكتبة Bookella لإدارة الكتب، الطلبات، العملاء، وعمليات الإعارة.

## 🚀 المميزات الرئيسية
- **إدارة الكتب والمخزون:** تتبع الكميات، أسعار الشراء والبيع، والربح المتوقع.
- **نظام الطلبات المتقدم:** تسجيل الطلبات، حساب الخصومات، وتتبع حالات الشحن.
- **إدارة المديونيات:** تتبع المبالغ المتبقية على العملاء وتسجيل الدفعات الجزئية.
- **نظام الإعارة:** تتبع الكتب المعارة وتواريخ الإرجاع مع تنبيهات للمتأخرات.
- **نقاط الولاء:** احتساب نقاط آلي للعملاء (نقطة لكل 10 جنيهات).
- **لوحة تحكم (Dashboard):** إحصائيات بيانية للأرباح والمبيعات ونقص المخزون.
- **واجهة عربية بالكامل:** تصميم عصري يدعم القراءة من اليمين لليسار (RTL).

## 🛠️ كيف تشغل المشروع على جهازك؟

لأن هذا المشروع يحتوي على "سيرفر" و"قاعدة بيانات"، فإنه يحتاج لخطوات بسيطة لتشغيله:

1. **تثبيت المتطلبات:**
   - تأكد من تثبيت [Node.js](https://nodejs.org/).
   - قم بتثبيت مدير الحزم pnpm: `npm install -g pnpm`.

2. **تحميل الكود:**
   - `git clone <رابط-المستودع>`
   - `cd bookella-system`

3. **تثبيت المكتبات:**
   - `pnpm install`

4. **إعداد قاعدة البيانات:**
   - المشروع يستخدم Drizzle ORM. تأكد من وجود ملف `.env` يحتوي على رابط قاعدة بيانات MySQL.
   - تشغيل الجداول: `pnpm db:push`.

5. **تشغيل المشروع:**
   - `pnpm dev`
   - افتح المتصفح على: `http://localhost:3000`.

## 🌐 كيف ترفع المشروع ليراه الجميع (الاستضافة)؟

**مهم جداً:** GitHub Pages لا يدعم هذا المشروع لأنه يحتاج إلى "سيرفر". لرفعه أونلاين مجاناً أو بسعر بسيط، اتبع الخطوات التالية:

### الخيار الأول: [Render.com](https://render.com/) (الأسهل)
1. قم بإنشاء حساب وربطه بـ GitHub.
2. اختر **New Web Service**.
3. اختر مستودع `bookella-system`.
4. الإعدادات:
   - **Build Command:** `pnpm install && pnpm build`
   - **Start Command:** `pnpm start`
5. أضف متغيرات البيئة (Environment Variables) مثل `DATABASE_URL`.

### الخيار الثاني: [Railway.app](https://railway.app/)
1. ممتاز جداً لعمليات الـ Full-Stack ويوفر قاعدة بيانات MySQL بضغطة زر.

## 📁 هيكلية المشروع
- `client/`: تحتوي على الواجهة الأمامية (React + Tailwind).
- `server/`: تحتوي على السيرفر والمنطق البرمجي (Express + tRPC).
- `drizzle/`: تحتوي على تعريفات جداول قاعدة البيانات.
- `shared/`: الأكواد المشتركة بين السيرفر والعميل.
